Simple Webhook Test

Test n8n Webhook

ใช้หน้านี้เพื่อทดสอบว่า n8n webhook ทำงานได้หรือไม่

Instructions

  1. เปิด n8n ที่ https://n8n.phukhieo.ac.th
  2. สร้าง workflow ใหม่
  3. เพิ่ม "Webhook" node เป็นตัวแรก
  4. ตั้งค่า Webhook:
    • HTTP Method: POST
    • Path: ai-chatbot (หรืออะไรก็ตามที่คุณต้องการ)
    • Authentication: None
  5. เพิ่ม "Set" node ตัวที่สอง
  6. ใน Set node ตั้งค่า:
    • name: response
    • value: Webhook received successfully!
  7. เพิ่ม "Respond to Webhook" node เป็นตัวสุดท้าย
  8. บันทึกและเปิดใช้งาน workflow
  9. กลับมาที่หน้านี้และคลิก "Test Webhook"
